vtFontAlign ​

  • demo

  • config

vue
<vt-font-align
  :config="configFontAlign"
  :data="data"
  @changeValue="changeValue"
></vt-font-align>
js
const configFontAlign = {
  type: "vtFontAlign",
  label: "对齐方式",
  verticalAlignKey: "verticalAlign",
  alignKey: "align",
};
const data = reactive({
  verticalAlign: "center",
  align: "left",
});
  • 属性
属性名说明类型可选
type组件的 keystring必填
labellabel 展示string必填
verticalAlignKey垂直布局样式 keystring必填
alignKey水平布局样式绑定的 keystring必填
  • 方法
方法名说明类型可选
changeValue(key, value)绑定的 key 的值发生变化，触发改事件Function非必填
